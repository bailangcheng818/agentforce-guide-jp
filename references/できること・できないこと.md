# できること・できないこと（Agentforce の境界・MECE）

ブレスト（何を作るか）を一緒に考えるときの**ものさし**。各アイデアを **✅できる / ⚠️準備が要る / ❌範囲外** に即タグ付けし、❌は**近い実現案に誘導**する。これで「無茶なエージェント」を作らせない。
※業務向けのやさしい言葉で。技術詳細は [道具カタログ.md](道具カタログ.md) / [役割分担マトリクス.md](役割分担マトリクス.md)。

## まず大前提（一言で）

Agentforce は **「会話を通じて、主に Salesforce の中の仕事を、“文章” と “決まった処理” で助けるAI」**。
だから **会話・文章・データの読み書き・決まった処理・外部呼び出し・人への引き継ぎ** が得意。
**絵や動画を作る／自由な画面やアプリを作る** ようなことは守備範囲の外。

## ✅ できることの「型」（これで全部・6つ）

どんな業務アイデアも、たいていこの6つ（の組み合わせ）に落ちる。会話の中で・文章中心の入出力で行う。

| # | できることの型 | かみ砕くと | 例 |
|---|---|---|---|
| 1 | **調べて答える** | 検索・ナレッジ・データ照会 | 過去の似た事例、規程Q&A、注文状況 |
| 2 | **文章を作る** | 要約・提案・メール・回答文の生成 | 商談サマリー、返信文、議事録 |
| 3 | **記録を作る・更新する** | レコードの作成・更新 | ケース作成、商談の更新、活動の記録 |
| 4 | **決まった処理を実行する** | あらかじめ用意した自動処理を起動 | 申請、承認、手続きフロー |
| 5 | **外部システムを呼ぶ** | 連携済みの外部APIを呼ぶ | 在庫照会、配送追跡（連携済み前提） |
| 6 | **人に引き継ぐ** | 有人対応へエスカレーション | 難しい問い合わせを担当者へ |

> ブレストのコツ：ユーザーのやりたいことを、この6つの**動詞**に翻訳する。複数を束ねるのもOK（例：1で調べて→2で文章化→3で記録）。

## ❌ はっきり範囲外（これは作れない＝別の手段を案内）

| 範囲外 | なぜ | 近い実現案（誘導先） |
|---|---|---|
| 画像・動画・音声・音楽の**生成** | エージェントは文章中心 | 「文章での説明・下書きまで」に置き換え |
| 3Dゲーム・リッチな画面・**自由なアプリUIの構築** | エージェントは会話UI | 「会話で操作・案内する」形に |
| Salesforce の外で動く**独立アプリ** | 提供チャネルの中で動く | 別の開発（範囲外）として切り出す |
| リアルタイム制御・機器操作 | そういう実行基盤ではない | 別システムへ（範囲外） |
| AIモデルの**学習・訓練** | 利用するだけ | 範囲外 |
| インターネットを**自由に閲覧** | 設定された検索範囲のみ | 「用意した検索/Web範囲」に限定 |
| **厳密さを保証する計算・最終確定**（金額確定など） | 生成AIは概算 | 「決まった処理（④）」に任せれば可 |

## ⚠️ 条件付きでできる（グレー＝①の準備が要る）

「できるけど、先に人が土台を用意する必要がある」もの。ここを**先に見極める**と後で詰まらない。

| やりたいこと | 要る準備（①人間） |
|---|---|
| 過去データ・資料を**検索**して答える | 検索の仕組み（レトリーバー/Data Cloud）＋テンプレの動作確認 |
| **外部システム**のデータを使う | 接続・認証（未接続を新規につなぐ作業は範囲外） |
| 独自の**複雑なロジック・計算** | 専用処理（Flow/Apex）の用意 |
| **お客様**向けに公開 | 専用利用者・問い合わせ窓口（チャネル）の設定 |

## 誘導のしかた（発散→収束のUX）

1. **歓迎**：まずアイデアを自由に出してもらう（「どんなことでも大丈夫です」）。
2. **タグ付け**：一つひとつを ✅ / ⚠️ / ❌ に分ける（声に出して）。
   - ✅ →「それはできます」。⚠️ →「できます。ただ先に〇〇の準備が要ります」。❌ →「それは範囲外ですが、近いところで〇〇ならできます」。
3. **収束**：「まず一番効くものを1つ」に絞り、6つの型に翻訳して**仕様書**へ（[../assets/設計メモ-テンプレ.md](../assets/設計メモ-テンプレ.md)）。
4. **差し戻し**：⚠️ の準備が未了なら①へ（[前提チェック.md](前提チェック.md)）。

> 目的は2つ：**無茶なエージェントを作らせない**こと、そして**実現できるところは気持ちよく任せてもらう**こと。
